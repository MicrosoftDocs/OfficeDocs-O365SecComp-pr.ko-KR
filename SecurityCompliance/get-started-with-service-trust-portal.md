# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Microsoft Service Trust Portal 시작

Microsoft 서비스를 신뢰 포털 다양 한 콘텐츠, 도구 및 Microsoft 보안, 개인정보 보호 및 규정 준수 사례에 대 한 기타 리소스를 제공 합니다.
  
## <a name="accessing-the-service-trust-portal"></a>서비스 신뢰 포털에 액세스

서비스를 신뢰 포털 컨트롤 및 그 안에 클라우드 서비스 및 고객 데이터를 보호 하는 프로세스의 Microsoft의 구현에 대 한 세부 정보를 포함 합니다. 일부 서비스를 신뢰 포털에서 리소스에 액세스 하려면 하면 해야 Microsoft 클라우드 서비스 계정 (Azure Active Directory 조직 계정 또는 Microsoft 계정)으로 인증된 된 사용자로 로그인 하 고 검토 하 고 동의 Microsoft 규정 준수 자료 (영문)에 대 한 비밀 유지 계약입니다.
  
**기존 고객 서비스를 신뢰 포털에 액세스할 수 있는** [https://aka.ms/STP](https://aka.ms/STP) 다음 온라인 구독 (평가판 또는 유료) 중 하나를 사용 합니다. 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > Azure Active Directory 계정에 연결 된 조직 전체 범위의 문서 및 규정 준수 관리자와 같은 기능에 액세스할 수 있습니다. 개인 사용 하기 위해 만든 Microsoft 계정 서비스를 신뢰 포털 콘텐츠에 대 한 액세스를 제한 됩니다. 
  
**새 고객 및 Microsoft 온라인 서비스를 평가 하는 고객**
  
새 계정 만들기 또는 평가판 계정을 만들 수는 STP에 대 한 액세스를 가져올 (테스트 계정에도 사용 되는) 다음 가입 양식 중 하나를 사용 합니다.
  
- 새로운 [Office 365 비즈니스 평가판 계정](https://go.microsoft.com/fwlink/p/?LinkID=507653) 또는 새 [Office 365 Enterprise 평가판 계정](https://go.microsoft.com/fwlink/p/?LinkID=698279) 등록

- 새 [Dynamics 365 테스트 계정](https://go.microsoft.com/fwlink/?LinkId=252780) 에 등록

- 새 [Azure 평가판 계정](https://go.microsoft.com/fwlink/?LinkId=722737)을 등록 합니다. 
    
무료 평가판을 또는 구독에 대해 등록 된 STP에 대 한 액세스를 지원 하기 위해 Azure Active Directory 사용 하도록 설정 해야 합니다.
  
## <a name="using-the-service-trust-portal"></a>서비스 신뢰 포털을 사용 하 여

서비스를 신뢰 포털 기능 및 콘텐츠 주 메뉴에서 액세스할 수 있습니다.
  
![서비스를 신뢰 포털 메뉴](media/STPMenus1.png)

다음 섹션에서는 주 메뉴에 있는 각 항목에 설명 합니다.
  
### <a name="service-trust-portal"></a>서비스 보안 포털

**서비스를 신뢰 포털** 링크 홈페이지를 표시 합니다. 홈페이지를 얻으려는 신속 하 게를 제공 합니다.

### <a name="compliance"></a>규정 준수

다음 규정 준수 관련 도구 및 리소스를 제공합니다.

- **감사 보고서** -Microsoft 클라우드 서비스에 대 한 독립 감사 및 분석 보고서의 목록이 표시 됩니다. 이러한 보고서와 같은 Microsoft 클라우드 서비스 데이터 보호 표준 및 준수 규정 요구 사항에 대 한 정보를 제공합니다.
  
    - 국제 조직 표준화 (ISO)
    - 서비스 조직 컨트롤 (SOC)
    - 국가 Institute 표준 및 기술 (NIST)
    - 연방 위험 및 권한 부여 관리 프로그램 (FedRAMP)
    - 일반 데이터 보호 규정 (GDPR)

  감사 보고서 및 **감사 보고서** 페이지의 링크를 하는 방법에 대 한 정보는 서비스를 신뢰 포털 홈페이지에도 표시 됩니다. 
  
- **규정 준수 관리자** -규정 준수 관리자는 추적을 할당 하 고 Dynamics 365 Office 365 등의 Microsoft 클라우드 서비스에 관련 된 조직의 규정 준수 활동을 확인 하는데 도움이 되는 워크플로 기반 위험 평가 도구 및 Azure 합니다. 규정 준수 관리자를 사용 하 여 클라우드의 공유 책임 모델 내에서 규정 준수를 관리 합니다. 자세한 내용은 [데이터 보호 및 Microsoft 클라우드 서비스를 사용 하는 경우 규정 요구 사항을 충족 하는데 규정 준수 관리자를 사용 하 여를](meet-data-protection-and-regulatory-reqs-using-microsoft-cloud.md)참조 하십시오.

### <a name="industries--regions"></a>산업 & 영역

Microsoft 클라우드 서비스에 대 한 업계 및 지역별 규정 준수 정보를 제공합니다.

- **산업** -정보에 대 한 및 Microsoft 클라우드 서비스 조직의 표준 및 이러한 업계에 대 한 규정 준수를 도움이 되는 방법에 대 한 특정 업계 방문 페이지를 제공 합니다.

    - Education
    - 금융 서비스
    - 정부
    - 상태
    - 제조
    - 정품

- **지역** -Microsoft 클라우드 서비스 법률을 준수 하 다양 한는 다양 한 국가 법적 의견을 제공 합니다. 특정 국가 오스트레일리아, 체코, 독일, 폴란드, 루마니아, 스페인, 및 영국을 포함 합니다.
  
### <a name="documents--resources"></a>문서 & 리소스

쉽게 하기 위한 방법을 Microsoft 클라우드 서비스 보안을 유지 데이터를 이해 하 여 규정 준수 목표를 충족할 수의 목표와는 다양 한 보안 구현 및 디자인 정보를 제공 합니다. 콘텐츠를 검토 하려면 **문서 & 자원** 페이지에서이 옵션 중 하나를 선택 합니다.

- 백서, Faq 및 방법 하-문서
- 규정 준수 가이드
- 펜 테스트 및 보안 평가
- 보안 점수
- Azure 보안 및 규정 준수 설계도
- 감사 된 컨트롤
  
### <a name="trust-center"></a>보안 센터

## <a name="my-library"></a>내 라이브러리

서비스를 신뢰 포털에서 문서 및 리소스를 내 라이브러리 페이지에서 제품 키를 추가 하려면 내 라이브러리 기능을 사용 합니다. 이렇게 하면 단일 위치에 관련 된 문서에 액세스 합니다.  내 라이브러리에 문서를 추가할 문서 오른쪽에 **...** 메뉴를 클릭 하 고 **라이브러리에 추가**선택 합니다. 하나 이상의 문서 옆에 있는 확인란을 클릭 한 다음 페이지의 위쪽에 **라이브러리에 저장** 을 클릭 하 여 여러 문서 라이브러리 내에 추가할 수 있습니다.

또한 알림 기능 전자 메일 메시지는 사용자가 보낸 사용자가 추가한 내 라이브러리에 문서를 업데이트 하는 Microsoft 때마다 되도록 내 라이브러리를 구성할 수 있습니다. 알림을 설정 하려면 내 라이브러리로 이동 하 고 **알림 설정**을 클릭 합니다. 알림 빈도 선택할 수 있으며 알림을 보낼 조직에서 전자 메일 주소를 지정할 수 있습니다. 전자 메일 알림을 업데이트 된 문서에 대 한 링크 및 업데이트에 대 한 간략 한 설명을 포함 합니다.

또한 알림 켜기 여부에 관계 없이 지난 30 일 동안 업데이트 된 내 라이브러리에서 문서를 파악 했습니다 note 합니다. 업데이트에 대 한 간략 한 설명은 도구 설명에 표시 됩니다. 

## <a name="starter-packs"></a>Starter 팩

Starter 팩은 특정 산업 분야에 대 한 Microsoft 클라우드 서비스에 대 한 설명서는 Microsoft curated 집합입니다. 현재, 서비스를 신뢰 포털은 금융 서비스 조직에 대 한 다음과 같은 세 starter 팩을 제공합니다. 이러한 starter 팩 도움말 조직 및 평가 하 고, 보안, 규정 준수, 및 Microsoft 클라우드에서 개인정보를 평가 하 고, 매우 규제 금융 서비스 산업에서 Microsoft 클라우드 서비스를 구현 하는 데 도움이 되는 지침을 제공 합니다.

- **평가 Starter 팩** -금융 서비스 조직에 대 한 Microsoft 클라우드의 초기 평가 위해 사용 합니다.

- **평가 Starter Pack** -후 평가 검사 목록과 및 기타 지침에에서 따라이 starter 팩 보안과 규정 준수, 개인정보 보호에 관련 된 위험을 평가 하는 조직입니다.

- **감사 Starter Pack** -이 starter 팩 감사 컨트롤 및 기타 도구를 사용 하 여 도움이 되는 방식으로 Microsoft 클라우드 서비스의 구현을 과정을 안내 하는 지침에 대 한 사용자 조직의 노출을 위험을 줄입니다.

이러한 starter 팩에 액세스 하려면 **서비스를 신뢰 포털 gt_ 산업 & 영역 gt_ 업계 솔루션 gt_ 금융 서비스로**이동 합니다. 열 수 또는 starter에서 다운로드 문서 팩 내 라이브러리에 저장 합니다.

와 관련 된 서비스를 신뢰 포털 또는 오류 포털을 사용할 때 발생 하는 방법에 대 한 질문 드립니다. 하면 수도 연락 하 여 질문 및 서비스 신뢰 포털 준수 보고서 및 신뢰 리소스에 대 한 의견 STP 페이지의 아래쪽에 사용자 의견 링크를 사용 하 여 합니다.
  
여러분의 의견은 저희에 게 매우 중요 합니다. 어떤 않았거나 선호 하지 않아서 하는 방법에 대 한 의견 보내기 페이지의 맨 아래에 의견 단추를 클릭 또는 추천 제품 또는 제품 기능을 향상 시키기 위한 있을 수 있습니다.
