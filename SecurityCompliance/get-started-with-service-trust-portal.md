# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Microsoft 서비스를 신뢰 포털 시작 하기

Microsoft 서비스를 신뢰 포털 (STP) 다양 한 콘텐츠, 도구 및 Microsoft 보안, 개인정보 보호 및 규정 준수 사례에 대 한 기타 리소스를 제공합니다. 또한 Microsoft의 온라인 서비스 및 온라인 서비스 유지 관리 하 고 표준, 법률과 규정을 준수와 같은 추적 조직을 도움이 되는 방법에 대 한 정보 독립적으로 제 3 자 감사 보고서에 포함 됩니다.
  
- 국제 조직 표준화 (ISO)
    
- 서비스 조직 컨트롤 (SOC)
    
- 국가 Institute 표준 및 기술 (NIST)
    
- 연방 위험 및 권한 부여 관리 프로그램 (FedRAMP)
    
- 일반 데이터 보호 규정 (GDPR)
  
## <a name="accessing-the-service-trust-portal"></a>서비스 신뢰 포털에 액세스

STP 컨트롤 및 그 안에 클라우드 서비스 및 고객 데이터를 보호 하는 프로세스의 Microsoft의 구현에 대 한 세부 정보를 포함 합니다. 일부 STP 자료에 액세스 하려면 Microsoft 클라우드 서비스 계정 (Azure Active Directory 조직 계정 또는 Microsoft 계정)으로 인증된 된 사용자로 로그인 하 고 검토 하 고 해야에 대 한 Microsoft 비밀 유지 계약에 동의 규정 준수 자료 (영문)입니다.
  
 **기존 고객은 STP에 액세스할 수** 에 [https://aka.ms/STP](https://aka.ms/STP) 다음 온라인 구독 (평가판 또는 유료) 중 하나를 사용 합니다. 
  
- **Office 365**
    
- **Dynamics 365**
    
- **Azure**
    
> [!NOTE]
> Azure Active Directory 계정에 연결 된 조직 전체 범위의 문서 및 규정 준수 관리자와 같은 기능에 액세스할 수 있습니다. 개인 사용 하기 위해 만든 Microsoft 계정 서비스를 신뢰 포털 콘텐츠에 대 한 액세스를 제한 됩니다. 
  
 **새 고객 및 Microsoft 온라인 서비스를 평가 하는 고객**
  
새 계정 만들기 또는 평가판 계정을 만들 수는 STP에 대 한 액세스를 가져올 (테스트 계정에도 사용 되는) 다음 가입 양식 중 하나를 사용 합니다.
  
- 새로운 [Office 365 비즈니스 평가판 계정](https://go.microsoft.com/fwlink/p/?LinkID=507653) 또는 새 [Office 365 Enterprise 평가판 계정](https://go.microsoft.com/fwlink/p/?LinkID=698279) 등록
    
- 새 [Dynamics 365 테스트 계정](https://go.microsoft.com/fwlink/?LinkId=252780) 에 등록
    
- 새 [Azure 평가판 계정](https://go.microsoft.com/fwlink/?LinkId=722737)을 등록 합니다. 
    
무료 평가판을 또는 구독에 대해 등록 된 STP에 대 한 액세스를 지원 하기 위해 Azure Active Directory 사용 하도록 설정 해야 합니다.

### <a name="service-trust-portal"></a>서비스 보안 포털

**포털을 신뢰 하는 서비스는** What's New를 포함 하는 STP 홈페이지 하는 STP에 대 한 섹션에서 사용 하 고 최신 업데이트에 자세히 설명 하는 규정 준수 관리자를 연결 합니다. 
  
### <a name="compliance-manager"></a>준수 관리자

[데이터 보호 및 Microsoft 클라우드 서비스를 사용 하는 경우 규정 요구 사항을 충족 하는데 규정 준수 관리자를 사용 하 여를](https://support.office.com/article/429e686f-d8a6-455e-a2b6-3791d763f000) 사용 하 여 Microsoft 클라우드 서비스를 사용 하는 경우 데이터 보호 및 규정 요구를 충족 하는 데 도움이 됩니다. 
  
### <a name="trust-documents"></a>보안 문서

서비스를 신뢰 포털에서는 쉽게 하기 위한 Microsoft 클라우드 서비스에서 데이터를 유지 하는 방법을 이해 하 여 규정 준수 목표를 충족할 수의 목표와 풍부한 보안 구현 및 디자인 정보에 액세스할 수 보안을 유지 합니다. 콘텐츠를 검토 하려면 메뉴에서 옵션을 선택 합니다.
  
 - Microsoft 클라우드 서비스 관련 한 규정 준수 데이터 보호 표준 및 포함 하 여 규정 요구 사항에서 보고 하는 평가 **감사 보고서** 독립적으로 감사를 제공 합니다. 
  
 - **데이터 보호** 다운로드, Microsoft Azure, Dynamics 365 및 Office 365를 작동 하는 방법에 대 한 정보에 대 한 문서를 신뢰를 제공 합니다. 
  
 - **Azure 보안 및 규정 준수 설계도** turn 키 보안 및 규정 준수 솔루션을 제공 하 고 지원, 클라우드 채택 및 규제 또는 제한 된 데이터를 사용 하는 고객에 대 한 사용률을 가속화 하는 업계 분야별의 요구에 맞게 조정 합니다. 
  
### <a name="regional-compliance"></a>국가별 준수

이 섹션에서는 Microsoft 클라우드 서비스를 렌더링 하는 법적 의견의 형태로 자주 지역별로 특정 준수 정보를 제공 합니다.
  
 - **체코** Microsoft 온라인 서비스 준수 체코 법률에서 법적 의견을 제공합니다. 
  
 - **폴란드** Microsoft 온라인 서비스 법률을 준수 하는 폴란드의 법적 의견을 제공합니다. 
  
 - **루마니아** Microsoft 온라인 서비스 법률을 준수 하는 루마니아의 법적 의견을 제공합니다. 
  
 - **스페인** Microsoft 온라인 서비스 법률을 준수 하는 스페인의 법적 의견을 제공합니다. 
  
### <a name="privacy"></a>개인 정보

이 사이트는 GDPR GDPR 책임을 기술 하 고 조직 측정값에 대 한 이해에 게 유용한 설명서의 특정 요구를 해결 하는 사용할 수 있는 Microsoft 서비스의 기능에 대 한 정보를 제공 합니다. Microsoft는 GDPR를 지원 하기 위해 보급 했습니다.
  
 - **GDPR: 시작** -관련 콘텐츠 및 사용할 수 있는 도구에 대 한 링크 된 포털 GDPR을 신뢰 하는 서비스와 관련 된 내용에 대해 홈페이지입니다. 
  
 - **데이터 제목 요청** -어떻게 Microsoft를 사용 하면 관련 설명서 및 도구에 대 한 링크와 데이터 주체 요청에 응답할 수 있습니다. 
  
 - **데이터 위반** -Microsoft를 감지 하 고 응답 개인 데이터를 위반 하 고 관련 설명서 및 도구에 대 한 링크와 GDPR, 아래에서 컨트롤러에 게 알리는 방법에 대 한 정보입니다. 
  
 - **DPIA** -Microsoft가 자신의 데이터 보호 영향 평가 의무를 충족 하는 조직에서는 사용 하는 방법에 대 한 정보입니다.
