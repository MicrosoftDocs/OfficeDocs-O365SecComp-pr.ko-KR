# <a name="partially-indexed-items-in-content-search-in-office-365"></a>Office 365의 콘텐츠 검색에서 부분적으로 인덱싱된 항목

Office 365 보안에서 실행 하는 콘텐츠 검색 &amp; 준수 센터 검색을 실행 하는 경우 예상된 검색 결과에 부분적으로 인덱싱된 항목 자동으로 포함 합니다. 부분적으로 인덱싱된 항목은 Exchange 사서함 항목 및 어떤 이유로 검색을 위해 인덱싱되 완전히 받지는 비즈니스 사이트에 대 한 SharePoint OneDrive에 있는 문서입니다. 부분적으로 인덱싱된 항목 파일 일반적으로 포함 되는 Exchange에서-인덱싱할 수 없는 파일 형식의-전자 메일 메시지에 첨부 되는 합니다. 항목 검색에 대 한 인덱싱할 수 없는 및 검색을 실행 하는 경우 부분적으로 인덱싱된 항목으로 반환 되는 다른 이유는 다음과 같습니다. 
  
- 파일 형식은 인식할 수 없는 이거나 인덱싱에 대 한 지원 되지 않습니다.
    
-  메시지 예: 이미지 파일, 유효한 처리기를 없이 첨부 파일 포함 이것은 부분적으로 인덱싱된 전자 메일 항목의 가장 일반적인 원인입니다. 
    
- 파일 형식 인덱싱은 지원되지만 특정 파일에 대해 인덱싱 오류가 발생했습니다.
    
- 전자 메일 메시지에 첨부된 파일이 너무 많습니다.
    
- 전자 메일 메시지에 첨부된 파일이 너무 큽니다.
    
- 파일이 타사 기술을 사용하여 암호화되어 있습니다.
    
- 파일이 암호로 보호되어 있습니다.
    
> [!NOTE]
> 대부분의 Office 365 조직 부분적으로 인덱싱되는 크기에 따라 볼륨 및 12% 미만으로 콘텐츠의 1% 미만 포함 합니다. 볼륨 및 크기의 차이점에 대 한 설명 더 큰 파일 완전히 인덱싱할 수 없는 콘텐츠를 포함 하는 더 높은 확률입니다. 
  
법적 조사에 대 한 조직 부분적으로 인덱싱된 항목을 검토 하려면 필요할 수 있습니다. 로컬 컴퓨터 또는 Office 365 고급 eDiscovery 사용 하 여 분석 결과 준비 하는 경우 검색 결과 내보낼 때 부분적으로 인덱싱된 항목을 포함할지 여부를 지정할 수 있습니다. 자세한 내용은 [Investigating 부분적으로 인덱싱된 되는 Office 365 eDiscovery의 항목을](investigating-partially-indexed-items-in-ediscovery.md)참조 하십시오.
  
## <a name="file-types-not-indexed-for-search"></a>검색 시 파일 형식이 인덱싱되지 않음

비트맵 등의 파일 또는 MP3 파일을 특정 유형의 콘텐츠를 인덱싱할 수를 포함 되어있지 않습니다. 결과적으로, 검색 서버를 Exchange 인덱싱 및 SharePoint 이러한 종류의 파일에서 전체 텍스트 인덱싱을 수행 하지 않습니다. 이러한 종류의 파일을 지원 하지 않는 파일 형식으로 간주 됩니다. 파일 형식에는 전체 텍스트 인덱스를 비활성화 기본적으로 또는 관리자가 있습니다. 지원 되지않는 및 사용할 수 없는 파일 형식으로 콘텐츠 검색에서 인덱싱되지 않은 항목으로 표시 됩니다. 앞서 설명한 것, 부분적으로 인덱싱된 항목 검색을 실행 하는 경우 검색 결과 집합에 포함 될 수,으로 로컬 컴퓨터를 검색 결과 내보내기 또는 고급 ediscovery 검색 결과 준비 합니다. 
  
지원되는 파일 형식 및 사용하지 않도록 설정된 파일 형식의 목록은 다음 항목을 참조하세요.
  
- **Exchange** - [Exchange 검색에서 인덱싱하는 파일 형식](https://go.microsoft.com/fwlink/p/?LinkID=386618)
    
- **Exchange** - [Get-searchdocumentformat](https://go.microsoft.com/fwlink/p/?LinkID=724037)
    
- **SharePoint** - [크롤링되는 기본 파일 이름 확장명 및 구문 분석 되는 SharePoint에서 파일 형식](https://go.microsoft.com/fwlink/p/?LinkID=404033)
    

  
## <a name="messages-and-documents-with-partially-indexed-file-types-can-be-returned-in-search-results"></a>메시지 및 사용 하는 문서 부분적으로 인덱스 된 파일 형식을 검색 결과에 반환 될 수 있습니다.

부분적으로 인덱싱된 파일 첨부 파일이 있는 모든 전자 메일 메시지 또는 부분적으로 인덱싱된 SharePoint 문서 마다 자동으로 부분적으로 인덱싱된 항목으로 반환 됩니다. 다른 메시지 또는 전자 메일 메시지의 **제목** 속성과 같은 문서 속성 및 문서에 대 한 **제목** , **만든이** 속성은 인덱싱된 되 고 검색 하는 사용 가능한 때문입니다. 예, "재무"에 대 한 키워드 검색 해당 키워드 또는 파일 이름 또는 문서 제목 전자 메일 메시지의 제목에 표시 하는 경우 부분적으로 인덱싱된 파일 첨부 된 항목 반환 됩니다. 그러나 키워드는 파일의 본문에만 표시 하는 경우 메시지 또는 문서 것 반환할 부분적으로 인덱싱된 항목으로 합니다. 
  
마찬가지로, 부분적으로 인덱싱된 파일 형식의 문서 및 부분적으로 인덱싱된 파일 첨부 파일이 있는 메시지를 인덱싱된 및 검색할 수는, 다른 메시지 또는 문서 속성을 검색 조건을 충족 하는 경우 검색 결과에 포함 됩니다. 메시지 본문에 보내고 받은 날짜, 보낸사람 및 받는 사람는 첨부 파일 및 텍스트의 파일 이름을 포함 하는 검색에 대 한 인덱스는 메시지 속성입니다. 검색을 위한 인덱싱된 문서 속성 작성 및 수정 된 날짜를 포함 합니다. 따라서 메시지의 첨부 파일을 부분적으로 인덱싱된 항목 수, 경우에 메시지가 포함 됩니다 일반 검색 결과에 다른 메시지 또는 문서 속성의 값이 검색 조건과 일치 하는 경우.
  
보안에서 검색 기능을 사용 하 여 검색할 수 있는 전자 메일 및 문서 속성의 목록에 대 한 &amp; 준수 센터 [키워드 쿼리 및 콘텐츠 검색에 대 한 검색 조건을](keyword-queries-and-search-conditions.md)참조 하십시오.
  

  
## <a name="partially-indexed-items-included-in-the-search-results"></a>검색 결과에 포함 된 부분적으로 인덱싱된 항목
<a name="unindexeditemsresults"> </a>

조직 식별 하 고 이러한, 포함 하는 기능 및 특정 조사와 관련이 있는지 여부를 확인 하려면 부분적으로 인덱싱된 항목에 대해 추가 분석을 수행 해야 할 수 있습니다. 앞에서 설명한 것 처럼 검색 되는 콘텐츠 위치에 부분적으로 인덱싱된 항목 예상된 검색 결과 함께 자동으로 포함 됩니다. 검색 결과 내보내기 또는 고급 ediscovery 검색 결과 준비 하는 경우 이러한 부분적으로 인덱싱된 항목을 포함 하는 옵션이 있습니다. 파일을 검색 결과 내보내기 (영문) 또는 고급 eDiscovery를 준비 하는 경우 부분적으로 인덱싱된 항목을 포함 하는 인식할 수 없는 형식, 암호화 된 또는 다른 이유로 인덱싱된 받지 된 항목을 포함 하는 옵션을 중 하나를 선택 합니다.
  
부분적으로 인덱싱된 항목에 대해 주의 다음에 유의 합니다.
  
- 콘텐츠 검색을 실행 하면 총 수와 (검색 쿼리에서 반환 된) 부분적으로 인덱싱된 항목의 크기에에서 표시 됩니다 세부 정보 창에서 검색 통계 "인덱싱되지 않은 항목"로 표시 된 대로.
    
- Exchange의 항목을 다운로드 하는 옵션을 선택 하는 경우 부분적으로 인덱싱된 Exchange 항목은 자신이 속한 디렉터리, 또는 개별 메시지도 각 사서함에 대 한 별도 PST 파일로 내보내집니다 검색 결과 내보내고 부분적으로 인덱싱된 항목을 포함 하는 경우 메시지입니다. 부분적으로 인덱싱된 SharePoint 항목 **Uncrawlable**라는 폴더를 내보냅니다.
    
- 검색 결과를 내보내려는 특정 콘텐츠 위치 또는 조직에서 모든 콘텐츠 위치 검색 했으므로, 검색 조건과 일치 하는 항목이 들어 있는 콘텐츠 위치에서 인덱싱되지 않은 항목만 내보냅니다. 즉, 검색 하는 결과가 없으면 사서함 이나 사이트에서 발견 되 면 다음 해당 사서함 또는 사이트에 있는 모든 인덱싱되지 않은 항목 없습니다 내보낼 수 있습니다. 이 대 한 설명 위치는 조직에서 많은에서 부분적으로 인덱싱된 항목 내보내기 (영문) 될 수 있습니다 내보내기 오류가 발생할 가능성을 증가 되 고 되 내보내기 및 검색 결과 다운로드 하는데 걸리는 시간을 늘립니다.
    
    ( **만 클릭 하 여 검색 결과 내보낼 때 인덱스 항목 검색에 대 한 모든 콘텐츠 위치에서 부분적으로 인덱싱된 항목 내보내기, (검색 쿼리에서 모든 키워드를 제거) 하 여 모든 항목을 반환 하도록 검색 구성 한 후 부분적 으로만 내보내려면 인식할 수 없는 형식이 있는 항목에는 암호화 또는 다른 이유로 인덱싱된 받지** **출력 옵션**아래).
    
- 검색 결과에서 모든 사서함 항목을 포함 하도록 선택 하는 경우 또는 검색 쿼리 키워드를 지정 하지 않고 또는 날짜 범위를 지정 하는 경우 부분적으로 인덱싱된 항목 부분적으로 인덱싱된 항목을 포함 하 여 PST 파일에 복사 되지 될 수 있습니다. 즉, 일반 검색 결과에 모든 부분적으로 인덱싱된 항목을 포함 하 여 모든 항목을 자동으로 포함 됩니다.
    
- 부분적으로 인덱싱된 항목을 미리볼 수 사용할 수 없습니다. 검색에 의해 반환 되는 부분적으로 인덱싱된 항목을 보려면 검색 결과 내보낼 해야 합니다.

## <a name="partially-indexed-items-excluded-from-the-search-results"></a>검색 결과에서 제외 부분적으로 인덱싱된 항목

항목은 부분적으로 인덱싱됩니다 하 고 검색 쿼리 조건 맞지를 하는 경우에 검색 결과에서 부분적으로 인덱싱된 항목으로 포함 되지 않습니다. 즉, 항목 검색 결과에서 제외 됩니다. 예, 가정해 보겠습니다 검색을 실행 하 고 모든 콘텐츠를 포함 하도록 할 것 이므로 모든 키워드 나 속성을 포함 하지 않습니다. 하지만 쿼리에 대 한 날짜 범위 조건을 포함 합니다. 부분적으로 인덱싱된 항목을 해당 날짜 범위를 벗어나는, 하는 경우에 부분적으로 인덱싱된 항목으로 포함 되지 않습니다. 날짜 범위는 검색 결과에서 부분적으로 인덱싱된 항목을 제외할 수 있는 효과적인 방법입니다.
  
마찬가지로, 검색 결과 내보낼 때 부분적으로 인덱싱된 항목을 포함 하는 경우 검색 결과에서 제외 된 부분적으로 인덱싱된 항목을 내보낼 수 없습니다.
  
EDiscovery 사례와 관련 된 쿼리 기반 보류를 만드는 경우이 규칙에 대 한 예외가입니다. 쿼리 기반 보류를 만드는 경우 모든 부분적으로 인덱싱된 항목 보류에 배치 됩니다. 검색 쿼리 조건에 맞지 않는 부분적으로 인덱싱된 항목 및 날짜 범위 조건을 외부 빠질 수 있는 부분적으로 인덱싱된 항목이 포함 됩니다. 만들기 (영문) 쿼리를 기초로 하는 방법에 대 한 자세한 내용은 보유 하 고에 대 한 [Office 365 보안 및 규정 준수 센터의 eDiscovery 사례](ediscovery-cases.md#step-4-place-content-locations-on-hold)에서 4 단계를 참조 합니다.

SharePoint 문서에 대 한 제한 인덱싱의 목록, [SharePoint Online에 대 한 검색 제한](https://support.office.com/article/7c06e9ed-98b6-4304-a900-14773a8fa32f)을 참조 하십시오.

## <a name="more-information-about-partially-indexed-items"></a>부분적으로 인덱싱된 항목에 대 한 자세한 내용
<a name="moreinfo"> </a>

- 앞서 설명한 것 처럼 메시지 및 문서 속성 및 해당 메타 데이터와 인덱스가 때문에 해당 키워드 인덱싱된 메타 데이터에 나타나는 경우 키워드 검색 결과 얻을 수 있습니다. 그러나 동일한 키워드 검색 키워드를 지원 하지 않는 파일 형식 항목의 내용에만 표시 하는 경우 동일한 항목을 반환 하지 않습니다 될 수 있습니다. 이 경우 항목 부분적으로 인덱싱된 항목으로 반환 됩니다.
    
- 부분적으로 인덱싱된 항목이 포함 되는 검색 결과에 고 있기 때문에 검색 쿼리 조건 충족 (제외 되지 않은) 하는 경우 다음 그렇지 않습니다 예상된 검색 통계에 부분적으로 인덱싱된 항목으로 포함 합니다. 또한 것은 포함 되지 않습니다 부분적으로 인덱싱된 항목이 포함 된 검색 결과 내보낼 때 합니다.
    
- 파일 형식 인덱싱은 지원 하 고 인덱싱되는, 있지만 있을 수 부분적으로 인덱싱된 항목으로 반환 될 파일을 포함 될는 인덱싱 또는 검색 오류입니다. 예, 매우 큰 Excel 파일을 검색 때문일 수 있습니다 부분적으로 성공 했습니다 (첫번째 4MB 인덱싱되), 하지만 다음 파일 크기 제한을 초과 하는 때문에 실패 합니다. 이 경우 것 같습니다 부분적으로 인덱싱된 항목으로 검색 결과 같은 파일 반환 됩니다.
    
- Microsoft 기술을 사용 하 여 암호화 된 첨부 파일 인덱스는 및 검색할 수 있습니다. 비 Microsoft 기술을 사용 하 여 암호화 된 파일을 부분적으로 인덱싱됩니다.
    
- S/MIME을 사용 하 여 암호화 된 전자 메일 메시지 부분적으로 인덱싱됩니다. 이 암호화 된 메시지 또는 첨부 파일 없이 포함 됩니다.
    
- IRM(정보 권한 관리)을 사용하여 보호된 메시지는 인덱싱되며 쿼리 매개 변수와 일치할 경우 검색 결과에 포함됩니다.
